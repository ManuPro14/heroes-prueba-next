# Superheroes App

Aplicación web desarrollada con **Next.js 14**, **TailwindCSS** y **TypeScript** como parte de una prueba técnica de Software Colombia.

## 🧩 Funcionalidad

- Mostrar una lista paginada de superhéroes desde una API pública.
- Visualizar los detalles de un superhéroe en un modal al hacer clic.
- Navegación por páginas de resultados sin recargar.
- Barra de búsqueda para filtrar superhéroes por nombre.
- Manejo de errores cuando un superhéroe no está disponible.

## 🚀 Tecnologías usadas

- Next.js 14 (App Router)
- React (useState, useEffect)
- TypeScript
- TailwindCSS

## ▶️ Cómo ejecutar el proyecto

### 1. Instalar dependencias (si es necesario)

```bash
npm install
# o
pnpm install
```

### 2. Ejecutar en modo desarrollo

```bash
npm run dev
# o
pnpm dev
```

La aplicación estará disponible en: [http://localhost:3000](http://localhost:3000)
